# Usage

## Start a capture

1. Launch SONAR.
2. Select a capture interface.
3. Click **Start**.

## Observe live analysis

- Packet events are processed in real time.
- Flow matrix updates as traffic is parsed.
- Graph updates reflect communication paths.

## Stop capture

- Click **Stop** to end capture cleanly.
- No new packet should be added after stop is confirmed.

## Export

- Export matrix/logs from the analysis view when needed.
