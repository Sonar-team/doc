## Pourquoi utiliser SONAR ?

SONAR est une application desktop open source de capture et d'analyse de flux réseau.  
Elle aide à cartographier les communications, identifier les zones d'ombre IT/OT et documenter un périmètre réseau dans le cadre d'un audit, d'un inventaire ou d'une investigation.
